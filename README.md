# Game Controls
Use the arrow keys or `W`,`A`,`S`,`D` keys on your keyboard to control the snake's direction:
- ↑ (Up) or `W` - Move Up
- ↓ (Down) or `S` - Move Down
- ← (Left) or `A` - Move Left
- → (Right) or `D` - Move Right

Others:

- To **Start or Restart** the game - Press `enter`
- To **Pause** the game - Press `esc`
